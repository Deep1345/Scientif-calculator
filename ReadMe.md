# Scientific Calculator (C++)

## Project Overview
This project demonstrates a simple Scientific Calculator built using C++.  
It is integrated with DevOps tools including **GitHub, Jenkins, Docker, and Ansible**.

The purpose of this project is to show a complete **CI/CD pipeline workflow** where code pushed to GitHub automatically triggers a Jenkins build.

---

## Features
The calculator supports the following operations:

- Square Root (√x)
- Factorial (x!)
- Natural Logarithm (ln x)
- Power (x^b)

---

## Technologies Used

- C++
- GitHub (Version Control)
- Jenkins (Continuous Integration)
- Docker (Containerization)
- Ansible (Deployment Automation)
